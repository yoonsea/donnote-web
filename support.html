자주 묻는 질문

돈노트는 무료인가요?
네, 돈노트는 무료로 다운로드하여 모든 핵심 기능(거래 기록, 자산 관리, 통계, iCloud 동기화)을 사용할 수 있습니다. 광고를 제거하고 개발자를 후원하시려면 프리미엄 구독을 이용하실 수 있습니다.
프리미엄 구독 가격은 얼마인가요?
월간 ₩2,500 또는 연간 ₩20,000 (월 환산 약 ₩1,667, 33% 할인)으로 제공됩니다. 프리미엄 구독 시 모든 화면의 광고가 제거됩니다.
여러 기기에서 데이터를 함께 사용할 수 있나요?
네, 동일한 Apple ID로 로그인한 모든 iPhone·iPad에서 거래·자산·카테고리가 iCloud를 통해 자동으로 동기화됩니다. 별도 계정 가입이나 설정이 필요 없습니다.
신용카드 할부 거래는 어떻게 입력하나요?
거래 추가 화면에서 결제수단으로 신용카드를 선택하면 할부 옵션이 표시됩니다. 개월수(2~24개월)와 연 이자율을 선택하면 회차별 거래가 매월 자동으로 생성됩니다. 원금균등 + 잔여원금 기준 이자 방식으로 정확하게 계산됩니다.
대출 원금 상환은 어떻게 기록하나요?
이체 거래로 입력하시면 됩니다. 보내는 곳에 은행 계좌, 받는 곳에 대출 자산을 선택하면 원금 상환으로 처리되며 대출 잔액이 자동으로 감소합니다.
자산 잔액은 어떻게 계산되나요?
거래를 입력하면 자산 잔액이 자동으로 갱신됩니다. 지출 시 자산 잔액이 차감되고, 수입 시 가산됩니다. 이체는 보내는 자산에서 차감되고 받는 자산에 가산됩니다. 거래를 수정·삭제하면 잔액도 자동으로 조정됩니다.
데이터를 백업하거나 내보낼 수 있나요?
설정 화면의 데이터 섹션에서 거래 내역을 CSV(Excel 호환)로, 전체 데이터를 JSON으로 백업할 수 있습니다. JSON 백업은 다시 복원 기능으로 가져올 수 있습니다.
Face ID나 암호로 앱을 잠글 수 있나요?
네, 설정 화면의 보안 섹션에서 Face ID·Touch ID 잠금을 활성화할 수 있습니다. 앱이 백그라운드로 전환될 때 자동으로 잠기며, 잠금 해제 전에는 민감한 정보가 가려집니다.
카테고리를 추가하거나 수정할 수 있나요?
설정 → 카테고리 관리에서 카테고리를 추가·수정·삭제·정렬할 수 있습니다. 이모지와 색상을 자유롭게 선택해서 나만의 카테고리를 만드세요. 거래 추가 화면에서도 즉시 새 카테고리를 만들 수 있습니다.
외화 거래도 기록할 수 있나요?
자산을 USD, JPY 등 외화로 설정할 수 있습니다. 환율은 자동으로 가져와 KRW 환산 표시에 반영됩니다. 거래 시점의 환율이 스냅샷으로 저장되어 과거 거래의 KRW 환산값은 변하지 않습니다.

서비스 이용 면책

돈노트는 개인 가계부 관리를 돕는 앱입니다. 본 앱은 투자 자문, 세무 자문, 회계 자문이 아니며, 재무 관련 의사결정의 근거로 활용되어서는 안 됩니다.

앱에서 제공하는 통계, 잔액 계산, 환율 환산, 할부 이자 계산 등의 정보는 사용자 입력값을 기반으로 한 참고용이며, 정확성을 보장하지 않습니다. 중요한 재무 결정 전에는 반드시 자격을 갖춘 전문가와 상담하시기 바랍니다.

구독 안내

돈노트는 광고 제거 등 추가 혜택을 제공하는 자동 갱신 구독(돈노트 프리미엄)을 선택적으로 제공합니다. 구독 시 다음 사항에 동의하는 것으로 간주됩니다:

  • 월간 ₩2,500 또는 연간 ₩20,000으로 제공됩니다.
  • 구독은 현재 구독 기간 종료 최소 24시간 전에 취소하지 않으면 자동으로 갱신됩니다.
  • 결제는 구매 확인 시 Apple ID 계정에 청구됩니다.
  • App Store 계정 설정에서 구독을 관리하거나 취소할 수 있습니다.

광고 안내

무료 사용자에게는 일부 화면에 Google AdMob 기반의 광고가 표시됩니다. 프리미엄 구독 시 모든 광고가 제거됩니다.

개인정보 처리방침

1. 수집하는 정보

돈노트는 사용자가 직접 입력하는 다음 정보를 처리합니다:

  • 거래 정보(금액, 카테고리, 메모, 날짜 등)
  • 자산 정보(이름, 종류, 잔액, 통화 등)
  • 카테고리 및 사용자 설정

계정 생성, 개인 식별 정보(이름·이메일·전화번호) 수집을 하지 않습니다.

2. 데이터 저장 위치

모든 데이터는 사용자의 기기와 사용자 본인의 iCloud 계정에 저장됩니다. 개발자는 사용자 데이터에 접근할 수 없으며, 외부 서버에 별도로 저장하지 않습니다.

3. iCloud 동기화

iCloud 동기화는 Apple의 CloudKit 인프라를 사용합니다. 데이터는 사용자의 Apple ID와 연결된 iCloud 비공개 데이터베이스에만 저장되며, 다른 사용자나 개발자가 접근할 수 없습니다.

4. 외부 서비스

다음 외부 서비스가 일부 기능에 사용됩니다:

  • 환율 API: 외화 자산의 KRW 환산을 위해 공개 환율 데이터를 조회합니다. 사용자 정보는 전송되지 않습니다.
  • 주가 API: 주식 자산의 시세를 가져오기 위해 공개 시세 데이터를 조회합니다. 사용자 정보는 전송되지 않습니다.
  • Google AdMob: 무료 사용자에게 광고를 표시하기 위해 사용됩니다. AdMob은 광고 개인화를 위한 식별자를 사용할 수 있으며, 이는 iOS의 추적 권한 설정에 따라 제어됩니다. 프리미엄 구독 시 광고가 표시되지 않으며 관련 식별자도 사용되지 않습니다.

5. 데이터 삭제

앱 내에서 개별 거래·자산을 직접 삭제할 수 있으며, 설정에서 "모든 데이터 삭제" 기능으로 전체 데이터를 삭제할 수 있습니다. 앱을 삭제하면 기기에 저장된 데이터는 제거되지만, iCloud에 저장된 데이터는 iCloud 설정에서 별도 삭제가 필요합니다.

6. 어린이의 개인정보

본 앱은 13세 미만 어린이를 대상으로 하지 않으며, 의도적으로 어린이의 개인정보를 수집하지 않습니다.

7. 정책 변경

본 개인정보 처리방침은 앱 업데이트와 함께 변경될 수 있으며, 변경 시 본 페이지에서 공지합니다.
