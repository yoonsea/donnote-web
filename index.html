지출·수입·이체부터 신용카드 할부까지. 자산 잔액이 자동으로 반영되는 깔끔한 가계부. iCloud로 모든 기기에서 동기화하세요.

App Store에서 다운로드

주요 기능

가계부에 필요한 모든 것을 한 앱에서

💸

간편한 거래 기록

지출·수입·이체를 한 화면에서 빠르게 입력하세요. 천단위 콤마 자동 표시로 입력이 편합니다.

💳

자산 통합 관리

계좌·현금·체크카드·신용카드·대출·주식까지. 모든 자산의 순자산을 한눈에 확인하세요.

🔄

자동 잔액 반영

거래를 기록하면 자산 잔액이 자동으로 갱신됩니다. 일일이 잔액을 계산할 필요가 없습니다.

📊

월별 통계 차트

도넛 차트와 카테고리별 분석으로 어디에 얼마를 썼는지 한 번에 파악하세요.

📅

신용카드 할부

2~24개월 할부 자동 생성과 이자 계산. 매월 청구 흐름을 정확하게 반영합니다.

☁️

iCloud 동기화

iPhone, iPad 모든 기기에서 거래와 자산이 자동으로 동기화됩니다. 별도 설정 불필요.

🔒

Face ID 잠금

민감한 금융 정보를 Face ID·Touch ID로 안전하게 보호하세요.

🏷️

카테고리 커스터마이징

이모지·색상으로 나만의 카테고리를 자유롭게 만들고 정렬하세요.

💾

CSV·JSON 내보내기

거래 내역을 엑셀 호환 CSV로, 전체 데이터를 JSON으로 백업할 수 있습니다.

이렇게 사용하세요

3단계로 시작하는 가계부

1

자산 등록

은행 계좌, 카드, 현금 등 보유한 자산을 등록하고 현재 잔액을 입력합니다.

2

거래 기록

지출·수입·이체를 빠르게 입력하면 자산 잔액이 자동으로 갱신됩니다.

3

통계 확인

월별 차트와 카테고리 분석으로 지출 패턴을 파악하고 예산을 조정하세요.
