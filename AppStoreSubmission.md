### Notes for Reviewer (영문)
```
DonNote is a personal finance / ledger app that runs entirely on-device with iCloud sync for the user's own account.

KEY POINTS:
1. No sign-up or login required. Users go straight to the app.
2. Data is stored locally and synced via CloudKit to the user's personal iCloud account only.
3. The developer maintains no backend server.

IN-APP PURCHASES:
The app offers an optional auto-renewable subscription (DonNote Premium) that removes ads:
  • com.yoonLab.DonNote.premium.monthly — ₩2,500 / month
  • com.yoonLab.DonNote.premium.yearly  — ₩20,000 / year

All core features (transaction logging, asset management, statistics, iCloud sync, biometric lock, exports) are FREE.

ADVERTISING:
Free users see Google AdMob native ads on certain screens (transactions, assets, inbox, statistics, asset detail, category drill-down). Settings screen is ad-free. Subscribers see no ads.

CLOUDKIT:
The app uses iCloud.com.yoonLab.DonNote container with cloudKitDatabase: .automatic. All sync is between the user's own devices — no other users have access.

THIRD-PARTY APIS:
  • Public exchange rate API (for foreign currency conversion display)
  • Public stock price API (for stock asset value display)
  • Google AdMob (for ads to free users only)
None of these receive personally identifiable information.

HOW TO TEST:
1. Launch the app — a default cash asset is auto-created.
2. Tap "+" to add a transaction. Select category, amount, payment method.
3. Tap a credit card asset to view its detail; add installment transactions.
4. Go to Statistics tab to see monthly donut chart.
5. Settings → Premium Subscription → tap to view paywall (StoreKit Configuration is included for local testing).

Thank you for your review!
```
